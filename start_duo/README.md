# START DUO

## Programmeren
Je bewerkt alleen `app.py`. Hierin zet je jullie namen en bewerk je de code. Dit is ook het enige bestand wat je hoeft in te leveren. 

## Programma draaien
Klik vanuit `app.py` op play. Je kunt gebruik maken van de `test_gegeven_lijst` functie om te kijken of de onderdelen werken zoals je verwacht. 

## Tests draaien (visible)
Ga naar de terminal (CTRL `; dit zit boven de tab)
Typ in de terminal het commando: 
pytest start_duo

Als de tests groen zijn, voldoet je algoritme aan de zichtbare eisen.
Er zijn ook onzichtbare testen die de docent gebruikt voor het testen.
